# Matcha Capital Interactive Prototype

This repository contains the interactive prototype for the Matcha Capital platform.

## How to View
1. Download the repository.
2. Open `prototype/index.html` in any modern web browser.
3. Use the **Matcha Hub** to navigate between Internal, Lender, and Admin views.

## Features
- Interactive financial dashboard.
- Document Vault with blockchain verification simulations.
- Marketplace for project discovery.
- Role-based navigation and permissions control.
